# 🚀 AI Interview Planner

An AI-powered full-stack web application that helps users prepare for technical interviews through personalized learning roadmaps, AI-generated interview strategies, and curated technical and behavioral questions. The platform analyzes your profile and target role to create a structured preparation plan tailored to your goals.

# 🚀 Getting Started

## Clone the Repository

```bash
git clone https://github.com/Neo-jhon/InterviewPlanner.git
```

## Navigate to the Project

```bash
cd InterviewPlanner
```

---

## Backend Setup

```bash
cd Backend
npm install
npm run dev
```

---

## Frontend Setup

```bash
cd Frontend
npm install
npm run dev
```

---

# 🔑 Environment Variables

Create a `.env` file inside the **Backend** directory.

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_jwt_secret

GEMINI_API_KEY=your_gemini_api_key
```
